# Taskify

A streamlined task management web application designed to help users organize their daily to-dos. This project features a secure user authentication system and a straightforward interface for tracking tasks from creation to completion.

##  Key Features

* **User Authentication:** Secure sign-up and log-in functionality to keep individual user data private.
* **Task Creation:** Authenticated users can quickly add new tasks to their personal dashboard.
* **Status Tracking:** Each task can be toggled between "Pending" and "Done" states for clear progress monitoring.
* **Personalized Dashboard:** Users only see and manage their own specific tasks.

##  Technology Stack

| Technology | Role |
| :--- | :--- |
| **PHP** | Backend logic, user session management, and routing. |
| **MySQL** | Relational database for storing user accounts and task records. |
| **HTML / CSS** | Frontend layout and visual styling. |
